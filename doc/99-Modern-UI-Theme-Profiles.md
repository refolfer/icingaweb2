# Modern UI Theme Profiles

## Available Variants

1. `NeoNOC`: dark baseline for general NOC work
2. `NeoNOC-Compact`: denser dark variant for high-cardinality tables and wallboards
3. `NeoNOC-Light`: light variant for bright office environments

## Selection Guide

- Choose `NeoNOC` when operators prefer contrast-heavy dark UI
- Choose `NeoNOC-Compact` when rows-per-screen is a priority
- Choose `NeoNOC-Light` when ambient light is strong or dark UI causes eye fatigue

## Recommended Rollout Pattern

1. Keep `NeoNOC` as default
2. Allow opt-in to `NeoNOC-Compact` and `NeoNOC-Light`
3. Compare pilot feedback per shift
4. Finalize default after 1-2 weeks of usage
